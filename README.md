# PCB_RASPI
MPPT solar charger controller with a DC/DC boost controller for Raspberry Pi and GSM with Peripheral outputs for GPIO
Design on Autodesk EAGLE.

This PCB is a power management controller for Solar pannels and Li-po batteries.
As 2 40pin GPIO headers for raspberry pi and gsm connection
As 5 I/O for fans, leds, buttons and sensors.
1 USB-A output for 5V/1A to provide power for Raspberry Pi.
